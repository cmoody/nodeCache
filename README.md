nodeCache
=========

No external npm modules installed

To run make sure you have installed http://nodejs.org/ globally.

To run
`
node index.js
`

Commands
--------
- get <key>
- set <key> <value> (errors if key doesnt exist)
- add <key> <value>
- increment <key> <amount>
- decrement <key> <amount>
- delete <key>
- empty
- quit
